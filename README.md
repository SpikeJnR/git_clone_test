# git_clone_test

Git_clone_test — проект на Vite с TypeScript, настроенный для разработки, сборки и деплоя на Vercel.

---

## Установка

Клонируйте репозиторий и установите зависимости:

```bash
git clone https://github.com/SpikeJnR/git_clone_test.git
cd git_clone_test
npm install
```
